# 🎮 Java Oyun Projesi (Swing Tabanlı)

Bu proje, Java kullanılarak geliştirilmiş kapsamlı bir masaüstü oyun uygulamasıdır.  
Oyun; karakter sistemi, savaş mekanikleri, envanter yönetimi ve grafiksel arayüz içerir.

## 🚀 Özellikler

- 🧙‍♂️ Karakter sistemi (Player, Enemy, Boss vb.)
- ⚔️ Savaş sistemi (BattleManager)
- 🔥 Element tabanlı hasar hesaplama
- 🎒 Envanter sistemi
- 🧪 Farklı item ve potion türleri
- 🛡️ Ekipman sistemi (silah, zırh vb.)
- 🎵 Oyun içi müzik sistemi
- 🖥️ Swing tabanlı kullanıcı arayüzü
- ❤️ Can ve mana barları
- 📦 Item slot ve tooltip sistemi
